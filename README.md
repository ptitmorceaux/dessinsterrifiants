# dessinsterrifiants

Pour compiler les étapes :
```bash
make etape1
make etape2
make etape3
make etape4
make etape5
```

> Permet de compiler toutes les étapes (de 1 à 5)
> *Il faut que tous les fichiers existent*
```bash
make
```

> Pour supprimer les fichiers `.out`
```
make clean
```

> Pour supprimer les fichiers `.out` et `.o`
```
make fclean
```
